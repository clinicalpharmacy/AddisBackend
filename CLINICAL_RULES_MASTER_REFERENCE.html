Comprehensive Guide for CDSS Rule Configuration

1. Logic Blocks

Type Logical meaning Clinical Use Case
all AND (Everything must match) Checking for a specific drug + specific lab level.
any OR (At least one must match) Checking if a patient is on any drug from a list of NSAIDs.
not NOT (Must not match) Checking if a patient is NOT pregnant or NOT on a drug.

3. Patient Data Facts (Attributes)

👶 Demographics & Reproduction

  • age: Age in years.
  • age_in_days: Precise age for infants/neonates.
  • is_pregnant: Boolean (true/false).
  • pregnancy_weeks: Gestational progress (1-42).
  • is_lactating: Breastfeeding status.

🌡️ Vitals

  • vitals.systolic / diastolic: Blood pressure.
  • vitals.heart_rate: Heart rate (BPM).
  • vitals.temperature: Body temp (°C).
  • vitals.oxygen_saturation: SpO2 level (%).

🧪 Common Lab Results (prefix with labs.)

  • creatinine / egfr / bun: Renal function metrics.
  • potassium / sodium: Electrolyte levels.
  • hemoglobin / platelet_count: Hematology levels.
  • alt / ast: Liver function enzymes.
  • blood_sugar / hba1c: Glucose control.
  • inr: Clotting/Warfarin monitoring.

4. Advanced Rule Examples

🏥 Ex 1: Pregnancy Contraindication (Critical)

Logic: IF Pregnant AND taking Accutane.

{ "all": [ { "fact": "is_pregnant", "operator": "==", "value": true }, { "fact": "medications", "operator": "contains", "value": "isotretinoin" } ] }
🥛 Ex 2: Lactation Safety Alert

Logic: IF Breastfeeding AND taking Metronidazole.

{ "all": [ { "fact": "is_lactating", "operator": "==", "value": true }, { "fact": "medications", "operator": "contains", "value": "metronidazole" } ] }
🧪 Ex 5: Renal Dosing (Complex)

Logic: IF eGFR < 30 AND (Gentamicin OR Tobramycin).

{ "all": [ { "fact": "labs.egfr", "operator": "<", "value": 30 }, { "any": [ { "fact": "medications", "operator": "contains", "value": "gentamicin" }, { "fact": "medications", "operator": "contains", "value": "tobramycin" } ] } ] }
🚫 Ex 16: Untreated Hypertension (NOT + ANY)

Logic: IF Systolic > 140 AND NOT taking ANY heart meds.

{ "all": [ { "fact": "vitals.systolic", "operator": ">", "value": 140 }, { "not": { "any": [ { "fact": "medications", "operator": "contains", "value": "lisinopril" }, { "fact": "medications", "operator": "contains", "value": "amlodipine" } ] } } ] }
🧪 Ex 18: Missing Monitoring Check (NOT)

Logic: IF taking Warfarin but NO INR lab result on file.

{ "all": [ { "fact": "medications", "operator": "contains", "value": "warfarin" }, { "not": { "fact": "labs.inr", "operator": "exists", "value": true } } ] }
